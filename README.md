**Inflation, Consumer Spending, Income, and Economic Conditions in the United States**

## Project Overview  

This repository contains the full pre-analysis setup for an integrated data analysis project focused on key United States macroeconomic indicators. The project integrates datasets related to inflation, consumer spending, disposable income, interest rates, unemployment, and consumer sentiment.

The purpose of this repository is to organize data, document sources, prepare datasets for integration, and establish a reproducible workflow prior to analysis.

This repository does **not** include findings, interpretations, or recommendations. It is intentionally focused on building a structured, validated, and reproducible data preparation pipeline.

---

## Research Focus  

The planned research focus is to examine how inflation-related measures compare with consumer spending while incorporating broader economic context.

- Inflation metrics serve as the central variables  
- Consumer spending serves as the primary outcome variable  
- Supporting datasets provide economic context  

No conclusions are made at this stage. All interpretation will occur after data validation and integration are complete.

---

## Why These Datasets Were Selected  

These datasets represent key components of the consumer economy:

- **CPI Overall** provides a broad measure of inflation  
- **CPI Food, Shelter, Transportation** provide category-level detail  
- **PCE Total** measures consumer spending  
- **PCE Price Index (PCECC96)** supports inflation comparison  
- **Disposable Income** provides income context  
- **Federal Funds Rate** provides monetary policy context  
- **Unemployment Rate** provides labor market context  
- **Consumer Sentiment** reflects household expectations  

This combination allows inflation and spending to be analyzed within a broader economic environment.
